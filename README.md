# Agri Intelligence Platform

An AI-powered agricultural intelligence system built using Remote Sensing, Deep Learning, and Generative AI.

## Features

### Land Cover Classification

* Swin Transformer
* EuroSAT Dataset
* 97.56% Validation Accuracy

### AI Agricultural Advisor

* Gemini 2.5 Flash Integration
* Agriculture-focused Question Answering
* Streamlit Interface

### Upcoming Features

* NDVI Analysis
* Crop Stress Detection
* Yield Forecasting
* Satellite Image Analytics
* Decision Support System
